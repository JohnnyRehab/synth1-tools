# Synth1 .sy1 → JSON Converter

Synth1 のパッチファイル (.sy1) を JSON 形式に変換するシンプルな Web アプリケーションです。

## 概要

Synth1 は長年にわたり大量のパッチ資産が存在しますが、  
`.sy1` フォーマットは扱いにくく、他シンセへの移行も困難です。

このツールは `.sy1` を JSON に変換し、  
将来的な解析・変換・移行を容易にすることを目的としています。

## 特徴

- 完全ローカル動作（アップロード不要）
- `.sy1` → JSON 変換
- パラメータ名変換対応
- JSONダウンロード対応
- 軽量（単一HTML）
